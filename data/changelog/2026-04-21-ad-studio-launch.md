A dedicated **Ad Studio** page where you create, upload, and manage all the creative for your campaigns.

- Grid view of every video, image, and audio file in the account, with previews and download buttons.
- "New video" generates an avatar talking-head from a script using the ElevenLabs voice + HeyGen avatar pipeline.
- "Upload" handles MP4 / MOV / WebM / PNG / JPG / GIF / WebP / MP3 / WAV up to 200 MB each.
- Filter by media type, search by filename or script text, click any tile to preview full-size.
- Generated videos automatically get an "AI" badge so you can spot them in the library.

Same `🎬 Video` button is still in the chat toolbar — both paths feed into the same library.
